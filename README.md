# PROPOSTA DE PROJETO: DESENVOLVIMENTO DE PADRÕES E SISTEMAS DE DADOS PARA SOCIOBIODIVERSIDADE NO ÂMBITO DO PROJETO GEF "ENTRE-CIÊNCIAS"

## 1. IDENTIFICAÇÃO DO PROJETO

**Título:** Implementação de Extensão Darwin Core e Sistemas de Dados para Sociobiodiversidade  
**Instituição Executora:** Jardim Botânico do Rio de Janeiro (JBRJ)  
**Coordenador:** [Nome do coordenador do JBRJ]  
**Período de Execução:** 24 meses (2025-2027)  
**Valor Total:** USD 110.000,00  

## 2. JUSTIFICATIVA

O projeto GEF "Entre-ciências" prevê a sistematização de dados sobre sociobiodiversidade em territórios de Povos Indígenas e Comunidades Tradicionais (PIPCTAFs). O JBRJ, como instituição de referência em biodiversidade e dados botânicos, possui expertise única para desenvolver os padrões técnicos necessários, especialmente considerando o desenvolvimento já iniciado da extensão Darwin Core para sociobiodiversidade.

O projeto original propunha a implementação do padrão internacional Plinian Core, porém o JBRJ já desenvolve uma extensão Darwin Core específica para dados de sociobiodiversidade que se alinha melhor com os objetivos do projeto e oferece maior compatibilidade com o ecossistema internacional de dados de biodiversidade (GBIF/SiBBr).

## 3. OBJETIVOS

### 3.1 Objetivo Geral
Desenvolver e implementar padrões internacionais de dados e sistemas de informação para registro, gestão e compartilhamento de informações sobre sociobiodiversidade no âmbito do Sistema Brasileiro de Informação sobre Biodiversidade (SiBBr).

### 3.2 Objetivos Específicos
1. Finalizar e implementar a extensão Darwin Core para dados de sociobiodiversidade
2. Desenvolver ferramentas de coleta e curadoria de dados especializadas
3. Capacitar equipes e parceiros nos padrões desenvolvidos
4. Participar ativamente no Grupo de Trabalho Técnico em Gestão de Dados (GT-D)
5. Apoiar as atividades formativas do programa FormarBio

## 5. EQUIPE E RECURSOS HUMANOS

### 5.1 Equipe Principal
- **Coordenador Geral**: Pesquisador sênior do JBRJ (contrapartida institucional)
- **Bolsista Etnobotânica** (40h/semana): Nível doutorado ou especialização

### 5.2 Perfil dos Bolsistas

#### Bolsista Etnobotânica
- **Formação:** Doutorado em Botânica, Etnobotânica ou áreas afins
- **Experiência:** Conhecimento tradicional, sociobiodiversidade, trabalho com comunidades
- **Competências:** Taxonomia vegetal, sistemas de conhecimento tradicionais, curadoria científica

#### Bolsista Informática na Biodiversidade
- **Formação:** Mestrado em Ciência da Computação, Sistemas de Informação ou Bioinformática
- **Experiência:** Desenvolvimento de sistemas, padrões de dados, APIs
- **Competências:** Darwin Core, banco de dados, desenvolvimento web, padrões internacionais
